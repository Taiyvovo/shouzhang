# 富文本标记

在 `TextElement` 的 `text` 字段中使用标记语法，实现行内样式变化。支持嵌套。

## 支持的标记

| 标记 | 效果 | 示例 |
|---|---|---|
| `**text**` | 粗体 | `**重点**` → **重点** |
| `*text*` | 斜体 | `*emphasis*` → *emphasis* |
| `__text__` | 下划线 | `__重要__` → __重要__ |
| `~~text~~` | 傍点（字符上方圆点） | `~~强调~~` → ︙强︙调︙ |
| `[c=#hex]text[/c]` | 文字颜色 | `[c=#e94560]红字[/c]` |
| `[s=N]text[/s]` | 字号变化 | `[s=20]小字[/s]` |

其中 `#hex` 为 6 位十六进制颜色，`N` 为数字字号。

## 使用方式

在模板的 `text` 字段中直接写标记：

```json
{
  "type": "text",
  "default": "今天天气**很好**，阳光从窗纱里透进来。\n\n[c=#4A90D9]蓝色[/c]的文字和[s=28]小号[/s]的文字都可以混排。\n\n*斜体* __下划线__ ~~傍点~~ 全部支持。"
}
```

## 换行

富文本中 `\n` 正常换行，标记可以跨行：每个段落的样式独立解析。

## 嵌套

标记支持嵌套（解析器递归处理）：

```
**粗体中带*斜体*也可以**
[c=#ff0000]**红字粗体**和*红字斜体*[/c]
```

## 注意事项

- `*` 只匹配不紧邻另一个 `*` 的单个星号。`**` 优先匹配为粗体，不会被拆为两个斜体
- `\\` 转义为字面反斜杠
- 未闭合的标记会被当作普通文字处理
- 傍点圆点渲染在每个字符上方居中位置

## 对比：普通文本 vs 富文本

```json
// 普通文本（自动换行，单一样式）
{
  "type": "text",
  "default": "今天天气很好。\n阳光从窗纱里透进来。",
  "style": { "font": { "family": "serif", "size": 36 } }
}

// 富文本（内嵌多风格）
{
  "type": "text",
  "default": "**今天天气很好。**\n*阳光从窗纱里透进来。*\n[c=#999]周日[/c] [s=24]随笔[/s]",
  "style": { "font": { "family": "Noto Sans SC", "size": 36 } }
}
```

富文本元素会被 `DocumentCompiler` 自动检测（检查是否包含 `**` `*` `__` `~~` `[c=` `[s=`），有标记就按富文本模式渲染，无标记走普通文本路径。
